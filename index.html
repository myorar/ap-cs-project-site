polygon generator

scratch.mit.edu/grace.polygongenerator

The goal of this project was to learn how to draw on scratch and experiment with allowing the user to input values. My partner and I went through a lot of trial and error until we discovered we could use mathematical formulas to code any shape. I learned how to run mathematical equations and create an accessible interface by creating this project.
